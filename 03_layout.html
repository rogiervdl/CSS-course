CSS layout

Overview

Layout overview

  • You'll use this in specific cases only:
    • position elements absolute, relative or fixed: CSS Positioning
    • float content around image or block: CSS Float
  • You'll use this for most of your layout:
    • one dimensional layout (menu, header items etc...): CSS Flexbox
    • two dimensional layout (page layout, item grids...): CSS Flexbox or Grid
  • for page layout, you can use Flexbox, or Grid, or both

CSS layout

Flexbox

About flexbox

  • Use Flexbox to arrange a list of items in a container in a row or column, with or without wrapping
  • The best tutorial on the web and beyond is this overview on css-tricks.com
  • Flexbox has CSS properties for the v, and for the individual items. In most cases, you'll be using just these box properties:
    container properties
    display: flex make this container a flexbox
    flex-flow: <row|column> <wrap|nowrap> arrange in a row or column, with/without wrapping
    justify-content: <flex-start|center|...> arrange items along main axis (row: horizontal, column: vertical)
    align-items: <flex-start|center|...> arrange items across main axis (row: vertical; column: horizontal)

When to use

  • CSS Flexbox is used for any one-dimensional layout, vertical or horizontal, with or without wrapping
  • You can use CSS Flexbox:
    • for horizontal menus
    • for creating columns, e.g. in the footer
    • for page layout
  • Don't use CSS Flexbox:
    • for fluid designs with floating elements where CSS floats are better
    • for two-dimensional grid-like layouts where CSS Grid is more appropriate

CSS layout

[Grid]

About Grid

  • Use Grid for two-dimensional layouts like item grids and page layout
  • The best tutorial on the web is this overview on css-tricks.com
  • In most cases, you're fine with just these properties:
    container properties
    display: grid make this container a grid
    grid-template-columns define grid column widths
    grid-template-rows define grid row heights
    grid-template-areas create named areas
    item properties
    grid-area: <area name> place item in area

    • for one-dimensional layout like menus wher Flexbox is better
