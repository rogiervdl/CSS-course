CSS layout

Overzicht

Layout technieken

  • CSS heeft een lange weg afgelegd; pas met het verdwijnen van Internet Exlorer (enkele jaren geleden nog maar!) kon Flexbox gebruikt worden, de eerste echte layout techniek in CSS.
  • Er bestaan vandaag 4 groepen layouttechnieken, elk met hun eigen toepassingsgebied:
    • flexbox: bedoeld voor alle soorten layout behalve waar één van de andere drie technieken geschikter is
    • floating: jaren misbruikt om een layout te forceren, maar nu enkel nog gebruikt waar het oorspronkelijk voor bedoeld was: content links of rechts rond een blok doen vloeien (bv. foto)
    • [grid]: moderne ietwat complexere methode bedoeld voor grid-achtige layout in rijen en kolommen
    • [positionering] (absoluut, relatief, fixed): maakt gebruik van coördinaten, en is daarom niet geschikt voor een modern flexibel responsive ontwerp; vandaag vooral gebruikt in Javascript games en uitzonderlijke gevallen als een back to top link
  • Flexbox is je standaard layoutmethode

Flexbox

Flexbox overzicht

  • Gebruik Flexbox om in een flex container een reeks flex children horizontaal of verticaal te schikken, met of zonder omloop (EN: wrapping)
  • De beste tutorial op het web is dit overzicht op css-tricks.com
  • Flexbox heeft aparte CSS properties voor de container en de children. Voor bijna alle gevallen volstaan deze vier properties:
    container properties
    display: flex maak deze container een flexbox
    flex-flow: <row|column> <wrap|nowrap> schik de children in een rij of kolom, met of zonder omloop
    justify-content: <flex-start|center|...> aligneer de children langs de hoofdas (rij: horizontaal, kolom: verticaal)
    align-items: <flex-start|center|...> aligneer de children langs de dwarsas (rij: verticaal; kolom: horizontaal)

  • Gebruik positionering:
    • voor HTML5 games waar je afbeeldingen moet kunnen stapelen
    • in zeldzame gevallen als een 'back to top' link
  • Gebruik positionering niet:
    • voor layout
    • om fouten te corrigeren

[Grid]

Grid overzicht

  • Gebruik Grid voor tweedimensionale layouts als item grids en pagina layout
  • De beste tutorial op het web is dit overzicht op css-tricks.com
  • In de meeste gevallen zit je goed met louter deze properties:
    container properties
    display: grid maak deze container een grid
    grid-template-columns definieer grid kolombreedtes
    grid-template-rows definieer grid rijhoogtes
    grid-template-areas creëer een gebied met een naam
    child properties
    grid-area: <area name> plaats een child in een gebied
